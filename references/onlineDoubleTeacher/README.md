# 在线双师课相关接口

在线双师课是标准课程下的一种双师课节类型（非普通课节）。默认不开通，如需开通，请联系客户经理。     

使用在线双师的API，需要注意
1. 本目录下的双师类API接口在双师功能和API均开通的时候才可以调用。
1. 双师课的主课节创建的时候为标准课节，请使用普通的创建课节和课堂API（非本目录下双师类API），在未绑定子课节之前视同普通课节。
1. 当标准课节作为主课节，使用 [创建在线双师课节](addClass.md) 绑定子课节后，它就变成一个双师课主课节，只能使用[编辑在线双师课节](editclass.md)和[删除在线双师课节](deleteClass.md)进行编辑和删除。无法再使用普通的课堂和课节的编辑接口了。
1. 如果主课节下的所有子课节都删除了，那它将恢复成一个普通课节。无法使用双师类编辑和删除接口。
1. 子课节只能用双师类接口创建、编辑和删除
1. 普通课节不能使用本目录下的接口。
