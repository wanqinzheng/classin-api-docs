## 机构订阅相关接口

对于老师和学生在教室里上课的行为数据，EEO会保存这些数据，并且提供消息订阅服务，以支持把数据推送给机构。

如果机构希望数据订阅，联系您的客户经理：
* 确定需要订阅教室内的哪些行为数据。具体行为数据分类，请访问[数据记录项](../datasub/description.md)；
* 提供一个数据接口地址，以接收来自EEO推送的数据；
* 提供您的机构SID；
* 提供一个或者多个邮箱，以便于接收消息订阅报错信息（通常为开发人员的邮箱）。

推送的数据，包括以下两个维度：
* 课节维度的数据：
  * 课节内实时推送的数据；
  * 课节结束后推送的数据；
* 机构维度的数据。
